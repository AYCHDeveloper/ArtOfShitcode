## The Principles

### 💩 Name variables in a way as if your code was already obfuscated

Good 👍🏻

```javascript
let a = 42;
```

Bad 👎🏻

```javascript
let age = 42;
```

### 💩 Always write comments in your local language

Good 👍🏻

```javascript
// Закриваємо модальне віконечко при виникненні помилки.
toggleModal(false);
```

Bad 👎🏻

```javascript
// Hide modal window on error.
toggleModal(false);
```
